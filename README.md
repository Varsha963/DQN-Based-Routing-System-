# DQN-Based-Routing-System-
## Overview
This project implements a Deep Q-Network (DQN)-based relay selection mechanism for efficient routing in Underwater Wireless Sensor Networks (UWSNs).

## Technologies Used
- Python
- TensorFlow
- NumPy
- Matplotlib
- Machine Learning
- Deep Reinforcement Learning

## Features
- Intelligent relay node selection
- Energy-efficient routing
- Improved packet delivery ratio
- Reduced transmission loss
- Enhanced network lifetime
